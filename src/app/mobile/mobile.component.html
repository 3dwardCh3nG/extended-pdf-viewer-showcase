The attribute `mobileFriendlyZoom` makes the toolbar larger, making it easier to use the PDF viewer on smartphones and tablet pcs. Usually it's enough to set it to 150%. Larger values are only useful if you reduce the number of buttons in the toolbar. [mobileFriendlyZoom]="'100%'" [mobileFriendlyZoom]="'150%'" [mobileFriendlyZoom]="'200%'" [mobileFriendlyZoom]="1.7" [mobileFriendlyZoom]="true" [mobileFriendlyZoom]="false"

The attribute "language" is only read when the component is drawn initially.

So this demo hides the PDF viewer for a second when you're switching from one language to another.

language default (usually English) es-ES (Spanish) de-DE (German) fr-FR (French) el (Greek) nl-BE (Belgian) ar (arab)

The attribute [useBrowserLocale] is a misnomer. It determines if the language files are loaded from the assets folder (two extra requests) or you define them somewhere in the DOM. The easiest way to do this is to add a <script type="application/l10n"> tag to the index.html.
